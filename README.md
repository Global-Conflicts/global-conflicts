## 1. Demo

Visit the demo website at: [global-conflicts.org](https://global-conflicts.org)

## 2. Setup

### 2.1 Frontend

Install npm and run:

```sh
cd frontend/
```

```sh
npm run dev
```

After that, the server is available under [localhost:3000](http://localhost:3000).
